## 第四步：配置 Nginx + HTTPS

```bash
# 安装 Nginx、Certbot（Ubuntu 示例）
sudo apt update
sudo apt install -y nginx certbot python3-certbot-nginx

# 复制项目内配置
sudo cp /opt/telegram-feishu-forwarder/deploy/nginx-99ka.life.conf \
        /etc/nginx/sites-available/99ka.life
sudo ln -sf /etc/nginx/sites-available/99ka.life /etc/nginx/sites-enabled/

# 先申请证书（按提示操作）
sudo certbot --nginx -d 99ka.life

sudo nginx -t && sudo systemctl reload nginx
```

验证：

```bash
curl https://99ka.life/feishu/health
# 应返回 {"status": "ok"}  （需先启动 listener）
```
